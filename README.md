# MensaReader

Web interface to OpenMensa data @ Bistro Frieden

## How to run

```bash
$ cd MensaReader
$ cpanm --notest --installdeps .
$ morbo --listen http://*:9000 mensa_reader
```

and open the website [`http://localhost:9000`][url] with your Browser.
